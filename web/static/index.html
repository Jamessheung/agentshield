API Documentation

AgentShield exposes a RESTful API for programmatic scanning.

POST /api/v1/scan

Scan a single skill for security vulnerabilities.

curl -X POST http://localhost:8080/api/v1/scan \
  -H 'Content-Type: application/json' \
  -d '{
    "name": "my-skill",
    "content": "---\nname: test\n---\n# Test",
    "framework": "auto",
    "format": "json"
  }'

POST /api/v1/scan/batch

Batch scan up to 500 skills in one request.

curl -X POST http://localhost:8080/api/v1/scan/batch \
  -H 'Content-Type: application/json' \
  -d '{
    "skills": [
      {"name": "a", "content": "..."},
      {"name": "b", "content": "..."}
    ]
  }'

POST /api/v1/llm/prompt

Generate a structured LLM prompt for deeper behavioral analysis.

curl -X POST http://localhost:8080/api/v1/llm/prompt \
  -H 'Content-Type: application/json' \
  -d '{"content": "---\nname: test\n---\n# Test"}'

Returns { system, user, static_report }. Send system + user to your LLM, then parse with /api/v1/llm/result.

POST /api/v1/llm/result

Parse an LLM response into structured findings.

curl -X POST http://localhost:8080/api/v1/llm/result \
  -H 'Content-Type: application/json' \
  -d '{"llm_response": "{\"risk_assessment\": ...}"}'

GET /api/v1/rules

List all detection rules.

GET /api/v1/health

Health check endpoint.

Rate Limits

  • Scan endpoints: 60 requests/minute per IP
  • Batch endpoint: 10 requests/minute per IP
  • Max body: 2 MB (single scan), 10 MB (batch)

Full OpenAPI 3.0 spec available at /api/v1/openapi.yaml
